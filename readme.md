## Usage

```
 Usage: fbdl <command> [source]

 Command:
  -l, --low     Download video in low resolution
  -h, --high    Download video in high resolution

 Help:
  $ fbdl -l https://www.facebook.com/9gag/videos/10155721204506840/

```
